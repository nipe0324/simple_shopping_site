simple_shopping_site
==============

This is Simple Shopping Site web application which is built by Ruby on Rails.

## Demo Site
[http://simple-shopping-site.herokuapp.com](http://simple-shopping-site.herokuapp.com)

## Ruby & Rails version
Ruby 2.0.0
Rails 4.0.0

## Database creation
To create database schema of this application.
```console
rake db:migrate
```

* Database initialization
To make user and product sample data.
```console
rake db:seed
```
